Info about this application

This application tracks the user's activities and records various statistics behind each activity. The user is able to view metrics about a chosen activity.
To run this application, download the project file and run it under the installed file directory with the command: $npm start. After the server is connected, go to this address: localhost:3000 to try out the application.
